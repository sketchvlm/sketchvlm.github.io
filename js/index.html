SketchVLM produces non-destructive SVG overlays to visually explain answers.

Abstract

When answering questions about images, humans naturally point, label, and draw to explain their reasoning...

SketchVLM

  • Visual scaffolding: optional coordinate grid for spatial reference.
  • Structured stroke prompting: XML stroke format for lines/curves/boxes/text.
  • XML → SVG conversion: renders an editable overlay on the original image.
